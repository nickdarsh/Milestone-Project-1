Project Introduction

This website serves as Project Milestone #1 for an IT course (IT 3203). The goal is to create a basic informational website to present research content using only HTML and CSS.

The site content focuses on the evolution of the Hypertext Transfer Protocol (HTTP) from 0.9 to 2.0.
